Rules of Puzzle Grid

  1. Fill the grid so each row contains each number exactly once.
  2. Each column must also contain each number exactly once.
  3. Each subgrid (bold separators) must contain each number exactly once.
  4. No guessing required in well-formed puzzles—use logic!
